* This Product Cipher implementation is based on the DES description at 
	http://page.math.tu-berlin.de/~kant/teaching/hess/krypto-ws2006/des.htm

* This application needs to be given a key of length 8 and messages of lenght a multiple of 8.

* output will be given as a binary string for a given valid Key.

* Given a encrypted output binary string wuth the key used for encryption, Decrypted text message will be displayed.
